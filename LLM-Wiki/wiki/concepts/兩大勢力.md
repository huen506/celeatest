# 兩大勢力

> 這不是善惡之爭。科技派的技術確實讓人活下去，自然派的土地確實在崩潰——兩件事同時為真、互相衝突，沒有人掌握完整資訊。

## 維拉斯聯席 Verath Consul（科技派）

| 項目 | 內容 |
|------|------|
| 核心信念 | 神明已失敗，技術才是秩序 |
| 手段 | 魔核研究、地脈採掘 |
| 據點 | 瓦倫克（工業核心）、德拉維克（採掘）、希洛葳（偏向） |
| 實質貢獻 | 天資不足者的真實救星，讓普通人得以生存 |
| 盲點 | 不知道地脈底層是瑪那殘流，正在加速大陸碎裂 |

## 利瓦恩 Livaern（自然派）

| 項目 | 內容 |
|------|------|
| 核心信念 | 世界需要癒合，而不是重建 |
| 手段 | 與自然共生，保育地脈 |
| 據點 | 艾爾凡（信仰核心）、德拉維斯（對外窗口） |
| 實質貢獻 | 天生魔力量高，能感知地脈危機 |
| 盲點 | 因果推論不完整，說不清楚危險在哪裡，所以雙方永遠無法真正對話 |

## 衝突的本質

兩邊都有**真實的理由**，也都有**真實的盲點**：

- 科技派：「我讓大家活下去」（真的）→ 但在燃燒世界的根基（也是真的）
- 自然派：「土地在哀鳴」（真的）→ 但解釋不清楚為什麼（推論有誤）

這是**資訊不對稱導致的結構性悲劇**，不是道德問題。

## 設計禁忌

- 不能簡化為善惡對立
- 科技派角色不知道自己在做壞事
- 自然派的部分判斷雖然感知正確，但推論錯誤
